# ST10395938\_PROG6212\_POE



\#  Contract Monthly Claim System (CMCS)



\## Overview

This web application allows lecturers to submit claims for hours worked, while Programme Coordinators, Academic Managers, and HR staff can verify, approve, and manage these claims. 





---



\## Features



\### Lecturer



\- Submit claims with:

&nbsp; - Hours worked

&nbsp; - Auto-calculated total based on hourly rate (pulled from HR records)

&nbsp; - Supporting documents (.pdf, .docx, .xlsx)

\- Track status of claims in real-time (`Pending`, `Approved`, `Rejected`)

\- Auto-populated fields for name, surname, and hourly rate

\- Upload multiple supporting documents with size/type restrictions



\### Programme Coordinator \& Academic Manager



\- View all pending claims

\- Approve or reject claims



\### HR (Super User)



\- Create and manage user accounts- Update user information 

\- Generate reports and invoices 

\- No self-registration; accounts are created and controlled by HR





\## Usage



1\. HR logs in and creates user accounts for lecturers and admins.  

2\. Lecturers log in, fill out claim forms, and upload supporting documents.  

3\. Coordinators and Managers log in to approve or reject claims.  

4\. HR can generate reports or invoices based on claim data.
